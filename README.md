# PitchPerfect
음성을 녹음하고 6가지의 음향효과와 함께 녹음된 파일을 재생하는 어플리케이션.

## 기본기능 - 녹음, 음향효과
- Slow
- Fast
- High Pitch
- Low Pitch
- Echo
- Reverb

## 나만의 kick
녹음된 파일 재생 중 UiSlider를 이용하여 재생위치를 움직일 수 있는 seek bar.
